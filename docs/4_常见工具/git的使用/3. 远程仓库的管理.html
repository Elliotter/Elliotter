1. 远程仓库的管理

1.1.1. 添加远程仓库

git remote add origin git@xxxxx.git

1.1.2. 本地同步远程

把本地库的内容推送到远程，用git push命令，实际上是把当前分支master推送到远程。

git push origin master

1.1.3. 克隆远程库

用命令git clone克隆一个本地库：

$ git clone git@xxxx.git
