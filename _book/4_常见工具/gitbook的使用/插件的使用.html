1. 配置与插件

介绍一下gitbook中book.json的一些实用配置和插件

1.1. 全局的配置

1.1.1. title

设置书本的标题

"title" : "Gitbook Use"

1.1.2. author

作者的相关信息

"author" : "mingyue"

1.1.3. description

本书的简单描述

"description" : "记录Gitbook的配置和一些插件的使用"

1.1.4. language

Gitbook使用的语言, 版本2.6.4中可选的语言如下： en, ar, bn, cs, de, en, es, fa, fi, fr, he, it, ja, ko, no, pl, pt, ro, ru, sv, uk, vi, zh-hans, zh-tw 例如，配置使用简体中文

"language" : "zh-hans"

1.1.5. links

在左侧导航栏添加链接信息

"links" : {
    "sidebar" : {
        "Home" : "https://www.baidu.com"
    }
}

1.2. 一些实用的插件

1.2.1. back-to-top-button 回到顶部

在book.json中添加以下内容。使用NPM安装（单独安装推荐NPM）npm install gitbook-plugin-back-to-top-button

{
    "plugins": [
         "back-to-top-button"
    ]
}

1.2.2. chapter-fold 左侧目录折叠

{
    "plugins": ["chapter-fold"]
}

1.2.3. expandable-chapters-small 左侧章节目录可折叠

{
    "plugins": [
         "expandable-chapters-small"
    ]
}

1.2.4. expandable-chapters 可扩展导航章节

{
    "plugins": [
         "expandable-chapters"
    ]
}

1.2.5. code 代码添加行号&复制按钮（可选）

{
    "plugins" : [ "code" ],
    "pluginsConfig": {
          "code": {
          "copyButtons": false
      }
    }
}

1.2.6. copy-code-button 代码块复制按钮

{
    "plugins": ["copy-code-button"]
}

1.2.7. search-pro高级搜索（支持中文）

{
    "plugins": [
         "-lunr", "-search", "search-pro"
    ]
}

1.2.8. emphasize为文字上底色

{
    "plugins": [
        "emphasize"
    ]
}

然后在markdown / asciidoc内容中，使用以下内容突出显示一些文本：

This text is {% em %}highlighted !{% endem %}
This text is {% em %}highlighted with **markdown**!{% endem %}
This text is {% em type="green" %}highlighted in green!{% endem %}
This text is {% em type="red" %}highlighted in red!{% endem %}
This text is {% em color="#ff0000" %}highlighted with a custom color!{% endem %}

1.2.9. splitter侧边栏宽度调节

{
    "plugins": [
        "splitter"
    ]
}
